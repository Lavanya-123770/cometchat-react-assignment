# CometChat Internship Assignment – React UI Kit Integration

This project is part of the CometChat Internship evaluation.  
It demonstrates the integration of the **CometChat React UI Kit** using the **UI Kit Builder**, customization, and evaluation of product experience.

---

##  Features Implemented

### ✔ Login Screen  
- Sample users (Andrew Joseph, George Alan, Nancy Grace, etc.)  
- UID-based login  
- Smooth authentication experience  

### ✔ Chats Module  
- One-on-one chats  
- Group conversations  
- Typing indicators (when supported)  
- Read receipts, delivery states  
- Message reactions ( etc.)

### ✔ UI Kit Builder Export  
The design was created & customized in the CometChat UI Kit Builder, exported as a React project, and integrated locally.

##  Findings & Issues Identified

### **Dashboard & Documentation Issues**
- Missing clarity on App ID vs API Key for beginners  
- Limited examples for real-world app flows  
- No onboarding tutorial for UI Kit Builder  
- No indication on where to paste credentials  

---

### **UI Kit Builder Issues**
- Tooltip load failure messages  
- No tooltips on icons (emoji, mic, attachments)  
- Theme inconsistency in dark mode  
- No user creation flow  
- Sidebar tabs sometimes unresponsive  
- Lack of group metadata preview  
- Alignment issues in date labels  
- No loading indicator during login  
- Hard to understand color configuration for beginners  

---

### **Implementation Issues**
- Blank chat screen — no empty state (e.g., “Start conversation…”)  
- Missing group description in header  
- No online/offline state for group members  
- Invalid UID gives no proper error message  
- Input icons lack hints  
- Auto-scroll inconsistent for first message
